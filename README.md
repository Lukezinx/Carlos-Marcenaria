# Carlos Marcenaria

Site institucional desenvolvido para a Carlos Marcenaria, especializada em móveis planejados em MDF.

## 🚀 Tecnologias

- React + Vite
- Tailwind CSS
- React Router DOM
- React Leaflet (mapa de área de entrega)
- React Icons
- Cloudinary (hospedagem de imagens)

## 📋 Funcionalidades

- Galeria de projetos com filtro por categoria
- Lightbox com navegação entre fotos
- Mapa interativo da área de entrega
- Menu responsivo com animações
- Botão flutuante do WhatsApp
- Design elegante com paleta dourada
